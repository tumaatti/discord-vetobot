# Discord-vetobot
Usage:

```
!help        - display this message
!bo<1/3/5>   - <user> <user> Start veto for best of 3 series
!vetostart   - <user> <user> ... Start veto with discord usernames
!vetostartv  - <voice channel name> Start veto with users in voice channel
!veto        - <map name> Veto on your turn
!vetostop    - Abort current veto
!ping        - Pong
```
